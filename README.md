# Welcome to Thundra Music

## Project info

**URL**: https://thundramusic.com

Thundra Music is a decentralized socio finance music-streaming platform that empowers artists, engages fans, and democratizes the music industry using Blockchain & AI. Our mission is to give artists control over their music, ensure transparent royalties, and reward fans with tokens for engagement, fostering a community-driven ecosystem.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
